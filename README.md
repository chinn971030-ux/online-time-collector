# 在线上课时间收集

这是可以发到企业微信里的在线共享版。老师创建一个收集链接，学生打开链接填写空闲时间，老师管理页面会自动汇总 1 小时课和 2.5 小时课的候补。

## 本地预览

```bash
npm start
```

打开：

```text
http://127.0.0.1:3000
```

## 分享方式

1. 打开首页，创建收集。
2. 进入老师管理页面后，复制“学生链接”。
3. 把学生链接发到企业微信群。
4. 学生填写并提交。
5. 老师管理页面点“刷新”，查看候补时间。

## 部署

这个版本没有外部依赖，可以部署到任何支持 Node.js 的平台，例如 Render、Railway、Fly.io、自己的服务器。

启动命令：

```bash
npm start
```

服务会读取平台提供的 `PORT` 环境变量。数据保存在 `data/sessions.json`。如果部署平台重启后会清空磁盘，请选择带持久磁盘的服务。
